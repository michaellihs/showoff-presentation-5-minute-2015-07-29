Twisted

Python Netzwerk Bibliothek

Was macht Twisted?

  • Bibliothek für Netzwerkprotokolle
  • Ermöglicht Schreiben von Client / Server Anwendungen
  • z.B. ssh-Server

Und was können wir damit tun?

  • ssh-Server "mocken"
  • z.B. für Deployment Tests

pip install twisted
pip install pyOpenSSL
pip install service_identity

Gitlab CLI

Kommandozeilen Tool für Gitlab

Was kann's?

  • Bedienung von Gitlab über CLI
  • Projekte anlegen
  • Merge Requests anlegen
  • Infos zu Projekten auslesen

Einrichtung

# In ~/.profile oder ...
export GITLAB_API_PRIVATE_TOKEN=\
  <PRIVATE GITLAB TOKEN>
export GITLAB_API_ENDPOINT=\
  https://gitlab.pluspunkthosting.de/api/v3

# Änderungen wirksamm machen
source ~/.zshrc

# Gem installieren
gem install gitlab

Gitlab CLI verwenden

gitlab create_project \
    'testkochbuch' \
    "{namespace_id: 105}"

Showoff

Präsentationen mit Markdown

Was ist showoff?

  • Sinatra Tool zum Erstellen von Präsentationen
  • Präsentationen werden in Markdown erstellt
  • Können sehr einfach auf Github publiziert werden

Vorteile

  • SCHNELLER!
  • Viele Syntaxhighlightings
  • Styling mit CSS (nicht mit der Maus...)
  • Schnelles Publizieren, öffentlich erreichbar
  • Präsentationsmodus

Nachteile

  • Kein vernünftiger PDF Export
  • "A little getting used to"
  • Bilder und Grafiken einfügen "suckt" etwas

Dokumentation

  • Etwas versteckt...
  • Projekt auf Github
  • Doku für Autoren

Showoff installieren

gem install showoff

Präsentation kickstarten

showoff init

Präsentation starten

showoff serve

# Dann erreichbar unter
# http://localhost:9090

Publikation auf Github

# Voraussetzung: Git Repository ist
# als git remote im aktuellen
# Repository

showoff github
git push origin gh-pages
